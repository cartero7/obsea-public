## Overview

OBSEA3 is a submarine power and instrument control system with ROS2 backend, FastAPI gateway, and React web interface.

**Key Features:**
- 🔌 12V/48V power rail control via I2C
- 🔐 JWT authentication with role-based access
- 📊 Real-time telemetry & monitoring
- 🚨 Safety limits with automatic protection
- 📱 Responsive web interface (desktop + mobile)
- 🔧 ROS2 integration for research workflows
